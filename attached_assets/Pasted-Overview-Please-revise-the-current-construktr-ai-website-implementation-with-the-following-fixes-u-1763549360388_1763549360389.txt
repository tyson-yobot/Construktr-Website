Overview:
Please revise the current construktr.ai website implementation with the following fixes, upgrades, and enhancements to better communicate the brand’s purpose and drive app downloads and user conversion. Please prioritize visual clarity, mobile optimization, high-impact copy, and visual polish.

1. 🧱 Foundational Fixes

A. Visual Contrast & Accessibility

Several areas use white or very light text (#ffffff or similar) over white/light-gray backgrounds (#fafafa or #f5f5f5), making the content unreadable.
✅ Update all such instances to use dark text (#1c1c1c or similar).
✅ Ensure minimum WCAG AA contrast for all text/background combinations.

Some backgrounds are too “blank” — visually empty white sections cause information fatigue.
👉 Switch to a branded gradient background in strategic blocks. Consider a clean light gray base (#f0f0f0) with subtle transitions into the construktr™ brand blue: #0A46E4 with a soft gradient blend (#e8eefc to #0A46E4).

Some tiles (especially features and pricing columns) appear misaligned or overly spaced. Use tighter grid spacing.
