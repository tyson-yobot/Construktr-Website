## SEO OPTIMIZATION CHECKLIST

For every blog post:

### Title Tag
- [ ] Primary keyword in first 60 characters
- [ ] Compelling + includes year or "Guide"
- [ ] Under 60 characters total

### Meta Description
- [ ] Primary keyword in first 100 characters
- [ ] Benefit-driven language
- [ ] Soft CTA at the end
- [ ] 140-160 characters

### H1
- [ ] Exact match or close variation of primary keyword
- [ ] Only ONE H1 per post

### H2s / H3s
- [ ] Include secondary keywords naturally
- [ ] Question format when possible (for featured snippets)
- [ ] Break up content every 200-300 words

### Content
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword 1-3 times total (0.8-1.2% density)
- [ ] Secondary keywords 1-2 times each
- [ ] Related semantic keywords throughout
- [ ] 900-1,200 words minimum

### Internal Links
- [ ] Link to 1 feature page (contextual anchor)
- [ ] Link to 1 solutions page (trade-specific)
- [ ] Link to pricing page (in CTA)
- [ ] Link to 1-2 related blog posts

### Images
- [ ] Alt text includes keywords
- [ ] File names include keywords (ai-estimating-screenshot.jpg)
- [ ] Compressed for fast loading
- [ ] At least 3-5 images per post

### Schema Markup
- [ ] Article schema with author, date, image
- [ ] FAQ schema if Q&A section included

### CTA
- [ ] Clear, benefit-driven
- [ ] Includes keyword when natural
- [ ] Links to trial signup or demo
