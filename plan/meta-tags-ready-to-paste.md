## IMPLEMENTATION NOTES

### To Add New Meta Tags:

1. Open `client/src/components/seo-head.tsx`
2. Add to the `pageMetadata` object
3. Use the exact format above
4. Ensure OG images exist or use a default

### OG Image Specs:
- Size: 1200x630px
- Format: JPG or PNG
- Include: CONSTRUKTR logo + feature/benefit headline
- No text smaller than 60px
- High contrast for social feed visibility

### Testing Checklist:
- [ ] Google search preview looks good
- [ ] Character counts: Title ≤60, Description ≤160
- [ ] OG image displays correctly on Twitter/LinkedIn
- [ ] Keywords appear natural in description
- [ ] Primary keyword in first 60 characters of title
